\# Journey Planner Application



\## Overview

A Python-based journey planner that processes structured transport data from Excel datasets to calculate travel distances and journey times between train stations.



\## Features

\- Reads and processes real-world transport data from Excel files

\- Computes journey distances and travel times between stations

\- Supports performance testing for evaluating efficiency



\## Technologies Used

\- Python

\- pandas 

\- Excel datasets

\- Handling and processing structured datasets

\- Applying algorithmic logic to real-world problems

\- Writing modular and testable Python code
